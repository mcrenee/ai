# AI 现金流投资全自动推演平台

## 项目概览
- **名称**: AI 现金流投资全自动推演平台
- **目标**: 提供专业的现金流投资分析和模拟工具
- **特性**: 
  - 📊 投资组合分析与评估
  - 💰 现金流推演与预测
  - 🎯 风险评级与压力测试
  - 📈 多维度投资指标可视化
  - 🔄 实时动态参数调整

## 已完成功能
- ✅ 投资组合概览仪表盘
- ✅ 现金流计算器
- ✅ 资产详情展示
- ✅ 风险评级系统
- ✅ 压力测试模拟
- ✅ 多场景对比分析
- ✅ 投资决策智能推荐

## 功能入口
- **主页**: `/` - 投资组合概览
- **所有功能**: 通过左侧导航栏访问
  - 投资组合概览
  - 现金流计算器
  - 资产详情
  - 风险分析
  - 压力测试
  - 场景对比
  - 智能推荐

## 数据架构
- **数据模型**: 纯前端 JavaScript 计算模型
- **存储服务**: 无需后端，所有数据在浏览器端处理
- **数据流程**: 用户输入 → 实时计算 → 动态展示

## 使用指南
1. 打开应用访问投资组合概览
2. 使用左侧导航栏切换不同功能模块
3. 在现金流计算器中调整投资参数
4. 查看实时计算的投资指标和预测结果
5. 使用压力测试评估不同市场条件下的表现
6. 参考智能推荐优化投资决策

## 本地开发
```bash

npm install -g http-server

# 启动本地服务器
npm run dev

# 访问 http://localhost:3000
```

## 部署到 Cloudflare Pages
```bash
# 部署到 Cloudflare Pages
npm run deploy
```
